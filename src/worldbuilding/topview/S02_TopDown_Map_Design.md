# S02: Ghost Protocol - Top-down Map 상세 설계

> **자신을 배신한 조직의 진실을 밝히는 네온 느와르 복수극**

## 설계 개요

**시나리오**: S02: Ghost Protocol  
**장르**: 네온 느와르 액션 RPG  
**주인공**: Kira "Ghost" Chen  
**환경**: 폭우 내리는 하층 빈민가 Sector 4 및 폐쇄된 지하철역  
**맵 스타일**: 수평적 다층 구조 (빈민가 → 지하 → 아파트)

---

## Zone별 상세 레이아웃

### Zone 1: The Forgotten Streets (빈민가 시장) - Sector 4

**테마**: 잊혀진 거리, 정보 수집, 느와르 분위기  
**페이싱**: Low Tension (탐색, 대화, 분위기)  
**플레이 타임**: 약 8~10분

#### 공간 구성

```
[진입 골목]
     ↓
[빈민가 시장] ← 중앙 광장
     ↓
[검문소 우회] (지붕 루트)
     ↓
[암시장 골목] ← POI: 정보 거래
     ↓
[지하철 입구] → Zone 2
```

#### 주요 요소

**1. 적 배치**

| 위치 | 유형 | 수량 | 패턴 | 난이도 |
|---|---|---|---|---|
| 중앙 광장 | 갱단 보초 | 3명 | 고정 위치, 대화 중 | ★☆☆ |
| 검문소 | 갱단 검문관 | 2명 | 직선 순찰, 10m 시야 | ★★☆ |
| 지붕 루트 | 스나이퍼 | 1명 | 회전 관찰, 15m 시야 | ★★☆ |

**2. 수집 아이템**

- **현상금 전단** (중앙 광장 벽): Kira의 수배 포스터 (내러티브)
- **상인 정보** (암시장): 지하철 비밀 통로 힌트 (비용: 100 크레딧)
- **소음기 업그레이드** (검문소 뒤 컨테이너): 소음 감소 +30%
- **응급 키트 x2** (시장 곳곳)

**3. 인터랙티브 오브젝트**

- **그래피티 표식** (Kira의 옛 마킹): 루트 안내 시각 단서
- **홀로그램 벚나무** (POI #1): 세이브 포인트 겸 허브
- **컨테이너 박스**: 엄폐물로 활용 가능
- **지붕 접근 파이프**: 파쿠르로 올라가 우회로 진입

**4. 동선 설계**

**정면 루트** (리스크):

```
중앙 광장 → 검문소 대화 / 뇌물 → 지하철 입구
예상 시간: 6분 | 난이도: ★★☆☆☆
```

**지붕 루트** (안전):

```
중앙 광장 → 파이프 등반 → 지붕 → 스나이퍼 회피 → 뒷문
예상 시간: 10분 | 난이도: ★★★☆☆
```

### Zone 2: The Subway (폐쇄된 지하철역)

**테마**: 지하 미로, 스텔스, 긴장감  
**페이싱**: Rising Action (긴장 고조, 타이밍 기반 스텔스)  
**플레이 타임**: 약 12~15분

#### 공간 구성

```
[지하철 입구 계단]
         ↓
    [개찰구] ← 잠김 (퓨즈 필요)
         ↓
   [발전실] → 퓨즈 획득
         ↓
    [플랫폼]
    ↙    ↓    ↘
[선로A] [선로B] [선로C]
   ↓     ↓     ↓
[Train 99] ← POI: 탈선 열차
         ↓
    [계단] → Zone 3
```

#### 주요 요소

**1. 적 배치**

| 위치 | 유형 | 수량 | 패턴 | 난이도 |
|---|---|---|---|---|
| 개찰구 | 청각 로봇 | 1기 | 고정, 소음 감지 | ★★★ |
| 플랫폼 | Scavenger 갱단 | 4명 | 불규칙 순찰 | ★★☆ |
| 선로 B | 엘리트 갱단 | 2명 | 좁은 경로 차단 | ★★★ |
| Train 99 내부 | 갱단 저격수 | 1명 | 내려다보는 위치 | ★★☆ |

**2. 메인 이벤트: 전력 복구 퍼즐**

| 단계 | 목표 | 장애물 |
|---|---|---|
| 1 | 발전실 진입 | 청각 로봇 회피 (걸어서만 이동) |
| 2 | 퓨즈 획득 | 암흑 속 장애물 (손전등 사용) |
| 3 | 개찰구 설치 | 시간 제한 15초 내 복귀 |

**3. 수집 아이템**

- **퓨즈** (발전실): 개찰구 해금 필수 아이템
- **Train 99 로그** (열차 내부): "이 열차가 멈춘 날..." (내러티브)
- **고압 전류 우회 도구** (선로 A): 전기 장애물 무력화
- **응급 키트 x2** (플랫폼 대기실)

**4. 환경 장애물**

- **고압 전류**: 선로 일부 구간, 접촉 시 즉사
- **깜빡이는 전조등**: 3초 주기로 빛/어둠 전환 (타이밍 스텔스)
- **무너진 천장**: 특정 경로 차단, 우회 필요

**5. 동선 설계**

**메인 루트** (선로 B):

```
개찰구 → 선로 B → 엘리트 회피 → Train 99 → 계단
예상 시간: 12분 | 난이도: ★★★★☆
```

**안전 루트** (선로 A + C):

```
개찰구 → 선로 A → 파이프 우회 → 선로 C → 계단
예상 시간: 15분 | 난이도: ★★☆☆☆
```

**숏컷** (Train 99 내부):

```
Train 99 진입 → 저격수 제압 → 위에서 플랫폼 건너뛰기
예상 시간: 8분 | 난이도: ★★★★★
```

### Zone 3: Confrontation (옛 은신처 아파트)

**테마**: 과거와의 대면, 좁은 공간 전투  
**페이싱**: High Tension (Climax) → Melancholy Release  
**플레이 타임**: 약 10~12분

#### 공간 구성

```
        [아파트 계단]
              ↓
    ╔═══════════════════╗
    ║   낡은 복도       ║
    ║                   ║
    ║  [101호] [102호]  ║
    ║                   ║
    ║  [103호] ← Kira의 방
    ╚═══════╦═══════════╝
            ║
    [비밀 금고실] ← POI #4
            ║
            ↓
      [보스 아레나]
   (방 + 베란다)
```

#### 주요 요소

**1. POI #4: The Safe Room (비밀 금고실)**

- **위치**: 103호 벽 뒤 숨겨진 공간
- **진입**: 벽의 특정 패널 상호작용 (비밀번호 힌트는 과거 사진 액자)
- **기능**:
  - **내러티브 보상**: 배신 단서가 담긴 홀로 메시지 재생
  - **장비 업그레이드**: 열화상 고글 (필수), 고급 방탄복
- **비주얼**: 먼지 쌓인 서버 랙, 작전 지도, 깜빡이는 홀로그램

**2. 보스전: 광학 위장 저격수**

| 페이즈 | HP | 패턴 | 대응 |
|---|---|---|---|
| Phase 1 | 100-70% | 베란다에서 저격 (3발씩) | 방 내부 엄폐물 활용 |
| Phase 2 | 70-40% | 광학 위장 + 근접 접근 | 열화상 고글로 위치 파악 |
| Phase 3 | 40-0% | 섬광탄 + 난사 | 베란다로 유인하여 역저격 |

**3. 아레나 구조**

```
╔═══════════════════════════╗
║  Kira의 옛 방 (103호)     ║
║                           ║
║  🗄️     💼     🗄️      ║ ← 엄폐물 (책상, 서랍)
║                           ║
║         ●                ║ ← 플레이어 시작
║      (시작 위치)          ║
║                           ║
║  [베란다 문] → ▓          ║
╚═══════════╦═══════════════╝
            ║
    ╔═══════════════╗
    ║   베란다      ║
    ║               ║
    ║   👹 BOSS    ║ ← 보스 초기 위치
    ║               ║
    ╚═══════════════╝
         (낙하 위험)
```

**4. 엄폐물 배치**

- **방 내부**: 책상 x2, 서랍장 x2 (파괴 가능, HP 200)
- **복도**: 벽 모서리만 활용
- **베란다**: 난간 (낮은 엄폐물, 저격당할 위험)

**5. 탈출 시퀀스**

보스 격파 후:

```
데이터 디스크 획득
    ↓
경찰특공대 진입 (30초 카운트다운)
    ↓
창문으로 탈출 (파쿠르 시퀀스)
    ↓
MISSION COMPLETE
```
