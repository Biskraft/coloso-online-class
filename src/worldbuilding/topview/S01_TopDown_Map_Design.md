# S01: Data Heist - Top-down Map 상세 설계

> **Apex Corp 타워 침투 작전: 100층 사이버펑크 잠입 미션**

## 설계 개요

**시나리오**: S01: Data Heist  
**장르**: 사이버펑크 잠입 액션  
**주인공**: Kira "Ghost" Chen  
**환경**: 폭우 내리는 밤의 Apex Corp 초고층 타워  
**맵 스타일**: 수직적 다층 구조 (지하~100층 옥상)

---

## Zone별 상세 레이아웃

### Zone 1: The Base (기단부) - 지하~1층

**테마**: 어둡고 습한 정비 구역, 튜토리얼 공간  
**페이싱**: Low Tension (탐색, 계획)  
**플레이 타임**: 약 5~7분

#### 공간 구성

```
[하수구 입구]
     ↓
[정화조 구역] ← 시작 지점
     ↓
[정비 통로 A] (드론 순찰 #1)
     ↓
[파이프 수직실] ← POI: 수직 이동 튜토리얼
     ↓
[환기구 진입점] → [Zone 2로 연결]
```

#### 주요 요소

**1. 적 배치**

- **위치**: 정비 통로 A
- **유형**: 보안 드론 x2 (저위험)
- **패턴**: 8자 순찰, 탐지 범위 5m, 순찰 주기 15초
- **대응법**: 타이밍맞춰 통과 or 해킹으로 무력화

**2. 수집 아이템**

- **PDA 로그 #1** (정비공의 메모): "33층 보안 업그레이드... 환기구는 구형"
- **응급 치료 키트** x1 (정화조 뒤편 숨겨진 구석)
- **해킹 툴 업그레이드**: 그래플링 훅 사거리 +2m

**3. 인터랙티브 오브젝트**

- **노란색 파이프** (그래플 포인트): 수직실 상단까지 연결
- **전력 박스**: 해킹하면 조명 꺼짐 (은신 용이)
- **환기구 덮개**: E키로 열어 Zone 2 진입

---

### Zone 2: The Spine (중층부) - 33층~75층

**테마**: 유리와 강철의 오피스 미로, 수직 이동  
**페이싱**: Rising Action (긴장 고조)  
**플레이 타임**: 약 12~15분

#### 공간 구성 (층별)

**33층 - 환기구 허브 (POI #1)**

```
[환기구 입구]
     ↓
[허브 중앙홀] ← 세이브 포인트
   ↙  ↓  ↘
[상층 루트] [아이템 루트] [비밀 루트]
   (빠름)    (안전)        (보상)
```

**50층 - 중간 오피스**

```
[대형 사무실] (카메라 x3, 경비 x2)
     ↓
[외벽 루트] ← 리스크 선택지
OR
[내부 복도] ← 안전 선택지
     ↓
[엘리베이터홀] (잠김)
     ↓
[비상계단] → [75층]
```

**75층 - 임원 구역 (POI #2)**

```
[고급 복도]
     ↓
[임원실] ← 옵션 탐험
   ↓
[비밀 금고] (광학 위장 업그레이드)
```

#### 주요 요소

**1. 적 배치**

| 층 | 유형 | 수량 | 패턴 | 난이도 |
|---|---|---|---|---|
| 33F | 감시 카메라 | 2기 | 120° 회전, 3초 주기 | ★☆☆ |
| 50F | 일반 경비 | 2명 | 직선 순찰, 15m 시야 | ★★☆ |
| 50F | 엘리트 경비 | 1명 | 불규칙 순찰, 청각 예민 | ★★★ |
| 75F | 적외선 레이저 | 그리드 | 고정, 닿으면 즉시 경보 | ★★★ |

**2. 수집 아이템**

- **데이터칩 #1** (33층): Apex Corp 내부 구조도 (+미니맵 확장)
- **응급 키트 x2** (50층 사무실 책상 서랍)
- **광학 위장 업그레이드** (75층 금고): 지속시간 5초 → 10초
- **이메일 로그** (75층): "Chimera 프로젝트... 위험하지만 수익성..."

**3. 동선 설계**

**메인 루트** (최단거리):

```
33F 허브 → 상층 환기구 → 50F 외벽 등반 → 75F 스킵 → 100F
예상 시간: 8분 | 난이도: ★★★★☆
```

**안전 루트** (추천):

```
33F 허브 → 아이템 루트 → 50F 내부 → 75F 탐험 → 100F
예상 시간: 15분 | 난이도: ★★☆☆☆
```

### Zone 3: The Brain (상층부) - 100층 서버룸

**테마**: 첨단 기술의 심장부, 메인 이벤트  
**페이싱**: High Tension (Climax)  
**플레이 타임**: 약 8~10분

#### 공간 구성

```
         [입구 케이블 통로]
                ↓
    ╔═══════════════════════╗
    ║   SERVER CORE ROOM    ║
    ║                       ║
    ║    [서버 랙 - A]     ║
    ║         ↑↓           ║
    ║  [CORE] ← 목표       ║
    ║    (중앙)            ║
    ║         ↑↓           ║
    ║    [서버 랙 - B]     ║
    ║                       ║
    ╚═══════════════════════╝
                ↓
         [비상 계단] → Zone 4
```

#### 주요 요소

**1. POI #3: Server Core (서버 코어)**

- **위치**: 100층 중앙
- **크기**: 반경 15m 원형 공간
- **특징**:
  - 바닥 유리(투명) → 100층 아래가 보임 (고소공포)
  - 푸른 빛기둥이 천장까지 관통
  - 냉각수 파이프 8개가 방사형으로 연결

**2. 메인 이벤트: 데이터 다운로드**

| 진행률 | 보안 대응 | 플레이어 대응 |
|---|---|---|
| 0-20% | 드론 x2 출현 | 엄폐 + 해킹으로 무력화 |
| 20-40% | 경비 x3 진입 | 광학 위장 + 이동 |
| 40-60% | 터렛 x2 활성화 | 터렛 뒤로 돌아 파괴 |
| 60-80% | 엘리트 x2 + 드론 x1 | 서버 랙 엄폐물 활용 |
| 80-100% | **보스: 사이버 워리어** | 패턴 회피 + 약점 공격 |

**3. 보스 패턴**

- **페이즈 1** (100-70% HP): 돌진 공격 + 권총
- **페이즈 2** (70-30% HP): 광역 EMP (모든 장비 3초 무력화)
- **페이즈 3** (30-0% HP): 광학 위장 + 근접 암살 시도

**4. 엄폐물 배치**

```
    서버 랙 A    서버 랙 B
       ■            ■
       
  ■                    ■
서버3               서버4

       [CORE 중앙]
       
  ■                    ■
서버5               서버6

       ■            ■
    서버 랙 C    서버 랙 D
```

- 각 서버 랙: 파괴 가능, HP 500
- 파괴 시 폭발 → 주변 5m 내 적에게 피해

---

### Zone 4: The Escape (옥상) - 100층 루프탑

**테마**: 개방된 탈출구, 해방감  
**페이싱**: Release (긴장 해소)  
**플레이 타임**: 약 2~3분

#### 공간 구성

```
       [비상 계단 출구]
             ↓
    ╔═════════════════╗
    ║  [헬리패드]     ║ ← Landmark
    ║     (원형)      ║
    ║                 ║
    ║  추격대 x5      ║
    ║                 ║
    ║  [탈출 지점]    ║ ← 끝까지 달려가기
    ╚═════════════════╝
          ↓
    [글라이더 전개]
    MISSION CLEAR!
```

#### 주요 요소

**1. 추격전**

- **적**: 특수 부대 x5 (중화기 장착)
- **목표**: 옥상 반대편 끝까지 도달 (직선거리 50m)
- **장애물**:
  - 헬리콥터 서치라이트 (회피 필요)
  - 낙하 위험 (난간 없음)

**2. 피날레 연출**

- 끝 지점 도달 → 자동 컷신
- 글라이더 펼치고 도시 야경 속으로 낙하
- 네온 불빛 사이를 비행하며 엔딩 크레딧
